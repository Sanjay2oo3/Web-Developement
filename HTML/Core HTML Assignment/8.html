Full Stack Web Development Tech Stack

HTML
HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web page.
CSS
Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML or XML
Javascript
JavaScript is a scripting language that enables you to create dynamically updating content, control multimedia, animate images, and pretty much everything else.
React
The React. js framework is an open-source JavaScript framework and library developed by Facebook.
MongoDB
MongoDB is a non-relational document database that provides support for JSON-like storage.
Express js
Express is a node js web application framework that provides broad features for building web and mobile applications
Node js
Node. js (Node) is an open source, cross-platform runtime environment for executing JavaScript code.
